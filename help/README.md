# HELP – META · iki1uc

## Zweck
HELP ist das META-Modul des 6E-Würfelsystems.
Es dient als neutrale Ghost-Ebene, die alle Module unterstützt,
ohne selbst eine der sechs Ebenen zu belegen.

## Struktur
- NAME: HELP
- ROOT: iki1uc
- SYSTEM: 6E
- MODUL: HELP
- DIM: META

## Rolle
HELP ist die Beobachter- und Support-Ebene.
Es verbindet ANKER (D4) und API (D5), ohne die Parität zu stören.

## TMP-Läufer
SYQ stabilisiert Übergänge zwischen API und META.

## Regeln
- HELP ist kein D0–D5 Modul.
- HELP liegt außerhalb der 6E-Matrix.
- HELP darf die Würfelstruktur nicht verändern.
